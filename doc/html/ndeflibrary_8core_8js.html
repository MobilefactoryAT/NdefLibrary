function initLibraryCore (   context)

Init wrapper for the core module.

Parameters
{Object}The Object that the library gets attached to in library.init.js. If the library was not loaded with an AMD loader such as require.js, this is the global Object.

This is the constructor for the Library Object. Please rename it to whatever your library's name is. Note that the constructor is also being attached to the context that the library was loaded in.

Parameters
{Object}opt_config Contains any properties that should be used to configure this instance of the library.
